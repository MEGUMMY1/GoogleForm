## 구글 설문조사 클론 코딩
https://clone-googleform.netlify.app

## 개발 언어
`React`, `TypeScript`, `Redux`, `Redux-toolkit`, `SCSS`

## 실행 방법
1. `npm install`
2. `npm start`
